# LINEスタンプ音声生成アプリケーション

画像をアップロードして、VOICEVOX音声付きのLINEスタンプを生成するWebアプリケーションです。

## 🎯 機能

- 画像アップロード（PNG/JPG形式、最大5MB）
- VOICEVOX音声合成
- 画像+文字のアニメーションGIF作成
- ZIP形式でのダウンロード

## 📋 必要な環境

- Python 3.8以上
- VOICEVOX（ローカルで動作）

## 🚀 セットアップ

### 1. 依存関係のインストール

```bash
pip install -r requirements.txt
```

### 2. VOICEVOXの起動

1. [VOICEVOX公式サイト](https://voicevox.hiroshiba.jp/)からダウンロード
2. VOICEVOXを起動（デフォルトでポート50021で動作）
3. ブラウザで `http://localhost:50021` にアクセスして動作確認

### 3. アプリケーションの起動

```bash
python app.py
```

4. ブラウザで `http://localhost:5000` にアクセス

## 📁 プロジェクト構造

```
├── app.py              # メインアプリケーション
├── requirements.txt    # 依存関係
├── README.md          # このファイル
├── templates/         # HTMLテンプレート
│   └── index.html     # メインページ
└── uploads/           # アップロードファイル（自動作成）
```

## 🎨 使用方法

1. 画像をアップロード
2. 音声にしたいテキストを入力
3. 「スタンプ生成」ボタンをクリック
4. ZIPファイルがダウンロードされます
5. 解凍して以下のファイルを使用：
   - `stamp_animated.gif`: LINEスタンプ用
   - `stamp_audio.wav`: 音声ファイル
   - `README.txt`: 使用方法説明

## 🔧 技術仕様

- **フロントエンド**: HTML + CSS + JavaScript（Vanilla）
- **バックエンド**: Flask
- **画像処理**: Pillow
- **音声合成**: VOICEVOX API
- **アニメーション**: imageio

## ⚠️ 注意事項

- VOICEVOXが起動していない場合、ダミー音声（無音）が生成されます
- ファイルサイズは5MB以下に制限されています
- 生成されたファイルは一時的に保存され、自動的に削除されます

## 🐛 トラブルシューティング

### VOICEVOXが起動しない場合
- VOICEVOXが正常にインストールされているか確認
- ポート50021が他のアプリケーションで使用されていないか確認
- ファイアウォールの設定を確認

### 画像アップロードエラー
- ファイル形式がPNG/JPGであることを確認
- ファイルサイズが5MB以下であることを確認

### 音声生成エラー
- VOICEVOXが起動していることを確認
- テキストが入力されていることを確認
